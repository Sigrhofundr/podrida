# English Description ğŸ‡¬ğŸ‡§

**Podrida Scorekeeper** is a free, open-source Android app designed to manage scores for the card game "Podrida" (a variant of Whist / Oh Hell).

Forget pen and paper! With this app, you can easily calculate scores, track dealers, and view live rankings.

## ğŸš€ Key Features

*   **Automatic Scoring**: Calculates scores automatically (10 points for correct bid + 3 per trick).
*   **Player & Dealer Management**: Supports 2 to 40 players, with clear indicators for the dealer and current turn.
*   **Multi-Language**: Full interface in **Italian**, **English**, and **Spanish**.
*   **Statistics**: Local history of games and leaderboards.
*   **Sharing**: Share final results directly to WhatsApp, Telegram, or other apps.
*   **Flexible Setup**: Choose between 40 or 52 card decks.
